# 📜 Quote Generator (React + Tailwind)

A simple, responsive quote generator built using **React** and **TailwindCSS**.

---

### ✨ Features

- 🎲 Fetches a random quote from a public API
- 📋 Copy to clipboard
- 🐦 Tweet the quote
- 🌀 Loading spinner while fetching
- 📱 Fully responsive layout
